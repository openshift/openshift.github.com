What is OpenShift Origin?

OpenShift Origin is the open source upstream of OpenShift, the next generation application hosting platform developed by Red Hat. Also known as Platform-as-a-Service, OpenShift takes care of infrastructure, middleware, and management so that you can focus on your app.

OpenShift Origin includes support for a wide variety of language runtimes and data layers including Java EE6, Ruby, PHP, Python, Perl, MongoDB, MySQL, and PostgreSQL.

Try it out

On your laptop

The easiest way to run OpenShift locally is to download an image suitable for running on a VM. The image will work on KVM, Virtual Box or VMWare.

You can also spin up a Virtual Box instance using Vagrant or build your own machine using Puppet

Online

You can try out OpenShift for free with OpenShift Online. Just sign up for an OpenShift Online account and create your first application in minutes.

Contribute

Extend OpenShift

You can add new runtimes and frameworks to OpenShift with community cartridges and quickstarts.

Cartridges provide fundamental features on OpenShift such as language runtimes and data layers. Community cartridges are contributed and maintained by the OpenShift community. You can read more about building cartridges here.

Quickstarts are codebases built specifically to run on OpenShift with one or more specific cartridges. A good example of a quickstart is the Ruby on Rails quickstart. It already has the appropriate OpenShift environment variables and hooks in place so that you can get started really easily without knowing much specifically about OpenShift. Building a quickstart is pretty easy, and a good place to start extending OpenShift. Read more about building quickstarts here.
